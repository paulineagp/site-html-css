Sobre o site

O projeto do site foi desenvolvido na disciplina de Fundamentos de Programação WEB. As tecnologias utilizadas no projeto incluem HTML, CSS e JavaScript. O site é composto por três páginas principais: Início, Sobre e Formulário. Cada uma dessas páginas foi projetada para atender a diferentes propósitos dentro do site.

Páginas Criadas:

  • Início: Esta é a página inicial do site, contendo informações básicas sobre a disciplina e a aluna.
  • Sobre: Esta página fornece detalhes sobre a construção do site e o propósito do projeto.
  • Formulário: Uma página de contato com um formulário que utiliza JavaScript para validar e enviar dados através do método GET.
